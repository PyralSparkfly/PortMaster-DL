# Guns of Fury

## Installation
Buy the game on Steam and add the data to `gunsoffury/assets`

https://store.steampowered.com/app/3099000/Guns_of_Fury/

## Default Gameplay Controls (Xbox Layout)
| Button            | Action                                |
|--                 |--                                     |
| START             | Menus                                 |
| SELECT            | Map                                   |
| D-PAD / JOYSTICK  | Move                                  |
| A                 | Jump                                  |
| B                 | Secondary Weapon                      |
| X                 | Primary Weapon                        |
| Y                 | Melee Weapon                          |
| L1                | Throw Weapon                          |
| R1                | Ability 1                             |
| L2                | Ability 2                             |
| R2                | Ability 3                             |
