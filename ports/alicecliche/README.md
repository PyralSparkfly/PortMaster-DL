## Controls

| Button | Action |
|--|--| 
|DPAD| Move |
|A| Talk / Examine |
|B| Attack|
|X (Hold)| Use stamina (runs, attacks faster)| 
|Y| Magic |
|R1| Next Level|
|L1| 1 Select Magic| 
|L2| 2 Select Magic| 
|R1| 3 Select Magic| 
|R2| 4 Select Magic| 

## Building

```
git clone https://github.com/matheusvilela/alice-in-clicheland.git
cd alice-in-clciheland
make
```
