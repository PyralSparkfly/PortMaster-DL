## Controls

| Button | Action |
|--|--| 
|DPAD|Movement|
|Left stick|Mouse movement|
|A|Left mouse button|
|B|Right mouse button|
|X|Auto play|
|Y|Change target in attack range |
|L1|Hotkey|
|L2|Slow mouse|
|R1|Stand still|
|R2|Swap DPAD into mouse mode|
|DPAD + Hotkey|Diagonal movement|
|A/B/X + Hotkey|Recite prayers|
